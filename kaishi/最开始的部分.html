C++ 的代码结构

                              //上一行是头文件
#define PI 3.14               //宏定义
using namespace std;          //命名空间
int abs(int a){               //自定义函数
    return a>0?a:-a;
}
int main(){                   //主函数
    int n;
    cin>>n;
    cout<<abs(n*PI);
    return 0;
}

4. OJ

这是Online Judge的简称。

中国主流的OJ有

  1. 洛谷中国最大的非校内OJ，没有之一。本套内容大部分习题会在洛谷上发布。
  2. Openjudge比较老牌的OJ，很多题目，推荐NOI，百练小组

国际上主流的OJ有

  1. Virtual judge，融合了很多OJ，可以进行跨平台评测，很强，很强。
  2. Codeforces，俄罗斯的网站，有点慢，比赛质量高，适合日常ACM训练。
  3. Atcoder，日本网站，比赛过水，适合打信心恢复赛，但是有难题。

不方便多啰嗦，请点开各自网站进行查看。
